02
Chapter two

Your pool is your power.

Your pool — your own package-locked stake plus delegator deposits — is scored against the admin-set tier curve. Each tier you climb lifts your share of the transaction fees collected by the whole network. APR comes only from the staking packages and accrues daily, claimable any time.

Illustrative staircase — the live tier curve is set by the chain admin
03
Chapter three

Delegators fuel you.

No hardware needed. Each validator markets its own packages — a monthly rate, a lock term, a minimum deposit and an early-exit penalty — on its public validator page. Delegators pick a package, send a join request, chat it out — and once accepted, deposit into that package. Every deposit counts toward the validator's tier directly and locks in the package's terms forever (grandfathered). Profit accrues daily and delegators withdraw themselves, any time — principal + profit (the package's penalty applies only before its lock ends).

Delegations flow in by APOS code — the fuller the ring, the higher the tier
Operators

Run an Elyon Chain node in 5 minutes.

The operator image — elyonchain/node — is pre-wired to this admin chain. Pull it, run one container, open port 3000, create or import a key. The node syncs itself, you apply as a validator with a stake, the admin approves, and your pool starts earning APR plus a share of transaction fees.

 Pull and run the operator image

Any host with Docker. Open TCP 3000 + 8545 + 30303 and UDP 30303 first.

docker pull elyonchain/node:latest

docker run -d --name elyonchain --restart unless-stopped \
  -p 3000:3000 -p 8545:8545 \
  -p 30303:30303 -p 30303:30303/udp \
  -v elyon-data:/data \
  elyonchain/node:latest

 Open port 3000 — create or import your key

Browse to http://YOUR_VM_IP:3000/ — the operator image lands you on the manager page, pre-wired to https://admintestnet.elyon.network. Create a new validator key (download the .txt and back it up) or import an existing one. The node starts, syncs, and the panel takes you to your dashboard.

 Apply as a validator, wait for admin approval

From the node panel, pick a staking package and apply as a validator — this stakes at least the larger of the network minimum and the package minimum into the APOS registry, locked for the package term. The admin approves with one click and you receive your public APOS code. From then on your tier (transaction-fee share) follows your pool volume, live in the validator panel — APR comes from your staking package. Keep your pool above the network minimum: if it drops below, earnings pause until it's refilled (then your validator is restored automatically).

 Attach your own domain with free HTTPS (optional)

Point an A-record at your VM, then run nginx-proxy + acme-companion once for an auto-renewing Let's Encrypt cert in front of port 3000.
